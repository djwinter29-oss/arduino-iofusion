# PlatformIO Arduino Uno Project

Basic PlatformIO project targeting the Arduino Uno (ATmega328P).

Build the project with PlatformIO (installed via `pip install platformio` or the VSCode PlatformIO extension):

```bash
cd /home/pi/arduino
pio run
```

Upload to a connected Uno (auto-detected serial port):

```bash
pio run --target upload
```
